## Contributing to this Project 

If you want to contribute to this work, please read the [Contributing Guidelines](CONTRIBUTING.md). There are multiple ways to contribute, including by opening issues, commenting in issues, and creating Pull Requests (PRs) to suggest new content or make editorial adjustments. 

We ask that potential contributors reach out to the CG before making any new content PRs by opening an issue or through the mailing list: [public-atag@w3.org](mailto:public-atag@w3.org). We welcome editorial PRs to address copy issues like spelling mistakes, broken links, or grammatical errors.
